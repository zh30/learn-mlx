## üõ†Ô∏è Why MLX?

[MLX](https://github.com/ml-explore/mlx) is an array framework for machine learning on Apple silicon, brought to you by Apple machine learning research. It‚Äôs designed by ML researchers for ML researchers, but it's powerful enough for everyone.

-   **Native to Mac**: Built specifically for the Metal GPU.
-   **Lazy Computation**: Only computes when needed, saving energy and memory.
-   **Unified Memory**: Efficiently shares memory between CPU and GPU.

## üèÅ Getting Started

### Prerequisites

-   A Mac with Apple Silicon (M1 or later).
-   macOS 13.5 or newer.
-   Python 3.10+ installed.

### Quick Setup

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/your-username/learn-mlx.git
    cd learn-mlx
    ```

2.  **Create a Virtual Environment**:
    ```bash
    python -m venv mlx_env
    source mlx_env/bin/activate
    ```

3.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
